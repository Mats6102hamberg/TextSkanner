# TextSkanner v1 – Demo Checklista

## Före demo
- [ ] `npm run dev` i `/frontend`
- [ ] Öppna http://localhost:3000

## Demo (3 min)

### 1. Maskering (30 sek)
- [ ] Klistra in text med personnummer/telefon
- [ ] Visa maskerat resultat → **$0 kostnad**

### 2. Dagbok (90 sek)
- [ ] Ladda upp bild på handskriven text
- [ ] Visa OCR-resultat
- [ ] Klicka "Klargjord" eller "Berättelse"

### 3. Avtal (60 sek)
- [ ] Ladda upp avtalsfil (PDF/bild)
- [ ] Visa sammanfattning + risker
- [ ] Peka på disclaimer: "Inte juridisk rådgivning"

## Avslut
- [ ] Nämn kostnadseffektivitet: gpt-4o-mini, regex först
